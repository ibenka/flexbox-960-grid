responsive flexbox 960 grid
================

Demo - http://kofeman.github.io/flexbox-960-grid/

How to use
==========

1) add css-file

`<link rel="stylesheet" href="flex960.css"/>`

2) write html-markup

`<div class="grid">`

  `  <div class="col1"></div>`
  
  `  <div class="col11"></div>`
  
`</div>`
